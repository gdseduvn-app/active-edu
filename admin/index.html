🚀 Xuất & Deploy lên GitHub

📦 Xuất dự án

Tải toàn bộ dự án ActiveEdu về máy dưới dạng file ZIP, sẵn sàng upload lên GitHub Pages.

Bao gồm: index.html, admin/, content/, assets/

📋 Hướng dẫn Deploy

  1. Tạo repo trên GitHub
  2. Upload toàn bộ file dự án
  3. Vào Settings → Pages
  4. Chọn branch main, folder / (root)
  5. Nhấn Save — site sẽ live sau ~1 phút

🔄 Đồng bộ Index

Tái tạo file content/index.json từ toàn bộ bài viết hiện có để cập nhật menu điều hướng.
