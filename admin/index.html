Đẩy Index lên GitHub

Sau khi tạo/xóa bài từ Admin, đẩy content/index.json mới lên GitHub để menu trang chính cập nhật.

Trạng thái đồng bộ

Nhấn Đồng bộ từ GitHub để kiểm tra.

Hướng dẫn

  1. Tạo bài mới → Lưu lên GitHub
  2. Admin tự cập nhật cây thư mục
  3. Nhấn Đẩy index.json để cập nhật menu
  4. Trang học sinh tự động nhận menu mới
