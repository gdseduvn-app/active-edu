✅ Đã cấu hình qua Cloudflare Worker

🔐 Tài khoản Admin
Mật khẩu admin được quản lý qua Cloudflare Worker
Để đổi mật khẩu, vào:
Cloudflare Dashboard → Workers & Pages → activeedu-proxy → Settings → Variables → ADMIN_PASSWORD

Sau khi đổi, đăng xuất và đăng nhập lại với mật khẩu mới.
