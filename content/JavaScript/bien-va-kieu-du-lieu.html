Biến trong JavaScript

Biến là nơi lưu trữ dữ liệu. Trong JavaScript hiện đại, ta dùng const và let:

const name = "ActiveEdu";    // Không thay đổi được
let score = 100;              // Có thể thay đổi
score = 95;                   // OK

Các kiểu dữ liệu cơ bản

  • String: "Xin chào", 'Hello'
  • Number: 42, 3.14
  • Boolean: true, false
  • Array: [1, 2, 3]
  • Object: { name: "Nam", age: 20 }
  • null / undefined

Ví dụ thực tế

const student = {
  name: "Nguyễn Văn Nam",
  age: 18,
  scores: [8, 9, 10, 7]
};

const avg = student.scores.reduce((a, b) => a + b) / student.scores.length;
console.log(`${student.name}: Điểm TB = ${avg}`);
