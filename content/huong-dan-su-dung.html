Giới thiệu ActiveEdu

ActiveEdu là nền tảng học tập trực tuyến miễn phí, có thể host trực tiếp trên GitHub Pages mà không cần server hay database.

Dành cho học sinh

  • Duyệt bài học theo thư mục từ menu bên trái
  • Dùng ô tìm kiếm để tìm bài nhanh
  • Nhấn vào thẻ thư mục ở trang chủ để mở rộng danh mục

Dành cho giáo viên / Admin

  1. Truy cập /admin/index.html
  2. Đăng nhập với tài khoản admin
  3. Tạo thư mục chủ đề mới trong Quản lý tệp
  4. Soạn bài học trong Soạn thảo
  5. Bài học tự động xuất hiện trong menu trang chính

Xuất bản lên GitHub Pages

Sau khi soạn thảo xong, vào Xuất & Deploy để tải file về và upload lên GitHub.

💡 Mẹo: Dùng Ctrl+S để lưu bài nhanh khi đang soạn thảo!
