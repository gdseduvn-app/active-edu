HTML là gì?

HTML (HyperText Markup Language) là ngôn ngữ đánh dấu chuẩn dùng để tạo các trang Web. HTML mô tả cấu trúc của một trang Web thông qua các thẻ (tags).

Cấu trúc cơ bản

Một trang HTML đơn giản có cấu trúc như sau:

<!DOCTYPE html>
<html>
  <head>
    <title>Tiêu đề trang</title>
  </head>
  <body>
    <h1>Nội dung chính</h1>
    <p>Đoạn văn bản.</p>
  </body>
</html>

Các thẻ quan trọng

  • <h1> đến <h6> — Tiêu đề các cấp
  • <p> — Đoạn văn bản
  • <a> — Liên kết hyperlink
  • <img> — Hình ảnh
  • <ul>, <ol>, <li> — Danh sách
  • <div>, <span> — Container
