CSS là gì?

CSS (Cascading Style Sheets) là ngôn ngữ dùng để mô tả cách hiển thị các phần tử HTML. CSS kiểm soát màu sắc, font chữ, khoảng cách, bố cục và nhiều hơn nữa.

Cú pháp CSS

selector {
  property: value;
  property: value;
}

Ví dụ:

h1 {
  color: #2563eb;
  font-size: 32px;
  font-family: 'Be Vietnam Pro', sans-serif;
}

p {
  line-height: 1.8;
  color: #444;
}

Ba cách áp dụng CSS

  • Inline: <p style="color:red">...</p>
  • Internal: Dùng thẻ <style> trong <head>
  • External: File .css riêng, link qua <link>

Box Model

Mỗi phần tử HTML được coi như một hộp với:

  • content — nội dung
  • padding — khoảng đệm bên trong
  • border — đường viền
  • margin — khoảng cách bên ngoài
div {
  padding: 16px;
  border: 1px solid #ddd;
  margin: 24px auto;
  border-radius: 8px;
}
